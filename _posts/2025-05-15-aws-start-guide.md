## Getting Started in the Cloud

I have a part time course ops role with the Univeristy of Michigan's Masters in Data Science program. 

One challenge I noticed a lot of teams encounter in their capstone project is storing and sharing large datasets and model objects. In the past we've seen teams send CSVs via slack, upload massive files to git, and some store their data in Google Drive. 

The program doesn't do a great job teaching cloud basics for data scientists, so I wrote the following series of modules to get students set up using AWS for their project.
