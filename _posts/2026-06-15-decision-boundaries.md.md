# Visualizing Decision Boundaries

This is a project where I break down how different kernels (Linear, RBF, Polynomial) affect the decision margins of an SVM. 

### Why this matters
When deploying models in production, understanding *why* a model classifies data points near the boundary layer prevents silent failures.

Here is a quick look at the core implementation:
* Built with Python and Scikit-Learn
* Visualized using Matplotlib contour plots
* Optimized via Cross-Validation grid search
